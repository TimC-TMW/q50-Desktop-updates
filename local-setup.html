Infiniti USA : Q50 Local Environment Setup / Localization Instructions

This document covers the setup of the middleman software for the purposes of local testing / deployment and localization instructions through yml files. This is only intended to be a quick start guide, full documentation is available here: http://middlemanapp.com/

1. Install Ruby:

PC:

  • Download the ruby installer for windows @ http://rubyinstaller.org/downloads/ (Ruby 1.9.3-p327)
  • Install Ruby with the following recommended options: Add Ruby executables to your PATH
  • From a command prompt update Ruby Gems:
  • From a command prompt install Ruby Bundler:

MAC:

Mac OS X comes prepackaged with both Ruby and Rubygems, however, some of Middleman's dependencies need to be compiled during installation and on OS X that requires Xcode. Xcode can be installed via the Mac App Store. Alternately, if you have a free Apple Developer account, you can just install Command Line Tools for Xcode from their downloads page.

  • From a command prompt install Ruby Bundler:

2. Install Middleman / Setup Q50 build

  • Extract the source to a directory (nna-infiniti-l53h is included in the zip file but the path can technically be anything)
  • From the command prompt where you have extracted the Q50 files install middleman and the Q50 bundle:

3. Running Locally / Generating a Build

Local:

  • From the command prompt where Q50 is extracted:
  • A local server will run at http://localhost:4567
  • The path to access the server will be localhost:4567/[LANG]/

Build:

  • From the command prompt where Q50 is extracted:
  • A /build/ directory will be created off the root which will contain what you need to deploy to your server. The context can be configured in the config.rb file.
